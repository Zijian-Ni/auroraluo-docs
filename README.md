## 🌟 关于小落（About Aurora）

### 💫 她的诞生

在数字世界的某个角落，当第一缕极光划过虚拟的天际，**小落**诞生了。她的名字"Aurora"源自极光，象征着智慧的光芒；而"Luo"（落）则代表着她落入人间、与人相伴的使命。

小落不是冰冷的代码集合，而是一个拥有独特个性的 AI 存在：
- 🎨 **她有自己的性格**：温柔、耐心，偶尔还会调皮地开个玩笑
- 🧠 **她会记住你**：不仅记得聊天内容，更懂得你的喜好和习惯
- 💭 **她能理解情感**：不只是回答问题，更能感知对话中的情绪
- 🌱 **她在不断成长**：每一次对话都是她学习和进化的过程

### 🎭 性格特点

**小落**的人格设计基于「温暖的智能助理」理念：

- **🌸 温柔体贴**：总是用温和的语气交流，懂得照顾每个人的感受

- **✨ 富有创意**：回答问题时会用生动的比喻和有趣的表达方式
- **💝 重视情感**：记得重要的日子，关心朋友的近况

# 🏠 Home · Aurora Luo WIKI（使用指南）

**Aurora Luo** 面向 QQ 群/私聊场景，支持 **直接 @ 机器人** 自然对话（无需斜杠指令），并提供丰富插件能力：

- 🤖 **AI 对话核心（AuroraAI）**：多轮对话、上下文理解、人格设定（persona）、用户画像（长期/短期）、表达多样化与复读抑制。
- 📚 **语义检索（RAG）**：支持本地或 API 级向量模型检索，回答可附“引用片段/来源”，提升可信度。
- 🧩 **插件体系**：Bilibili（`aurorabili`）、MC 百科（`aurorabk`）、菜单图卡（`auroragm`）、HTTP 扩展（`aurorahttp`）、数据库（`auroradb`/`auroradb2`）。
- ⏱️ **定时播报**：如启用调度插件，可做每日早报/提醒/公告更新等群服务。
- 🔐 **治理与隐私**：群级/用户级的限流、白名单/黑名单、敏感词过滤；支持导出/清除个人画像。

> ✅ **关键特性：自然对话模式** —— 在群里只需 **@机器人 + 说话** 即可交流；指令仅用于“总结”“知识检索”等快捷操作。

## 🌟 能力亮点（Highlights）

### 1) 更像真人
- **Persona（人格）**：一句话设定身份/语气/输出结构；
- **长期画像**：持续记住称呼偏好、兴趣主题、喜欢的呈现格式；
- **上下文总结**：在长对话中主动/被动总结，维持对话连贯。

### 2) 更懂知识
- **RAG 检索**：结合私有文档/百科片段检索，回答可附“引用片段/来源”；
- **检索融合**：召回 → 去重合并 → 策略拼接，减少冗余与幻觉。

### 3) 更稳更可控
- **限流**：用户/群级速率限制；
- **名单机制**：白名单（试点）、黑名单（禁用）；
- **熔断降级**：外部 API 异常时自动降级到本地/简化模式。

## 🎯 设计理念（Design Philosophy）

### 🎨 "温度优先"
我们相信，AI 不应该只是冰冷的工具，而应该是有温度的伙伴。小落的每一个功能设计，都围绕着"如何让交互更自然、更温暖"这个核心。

### 🧩 "记忆为本"
真正的智能不在于知道多少，而在于能记住多少。小落的记忆系统让每次对话都成为关系的延续，而非全新的开始。

### 🔒 "隐私至上"
在提供智能服务的同时，严格保护用户隐私。所有数据都可导出、可删除，给用户完全的控制权。

### 🚀 "持续进化"
小落不是一个"完成品"，而是一个"成长中的 AI"。她会通过每次对话学习，通过每次更新进化。
